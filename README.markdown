# [Bombyard][]

Bombyard is a clone of the minesweeper game in Haskell.

1.  Install [Stack][].

2.  `stack build`

3.  `stack exec bombyard`

![Screenshot][]

Click on a square to reveal it. Once a game is over, click anywhere to restart.
Press `esc` to quit.
